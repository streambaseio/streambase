StreamBase

StreamBase is an independent blockchain network, built with support from the UDP/Datagram connection method to enhance transmission capabilities.
The project implements a Proof-of-Stake (PoS) consensus model that is intentionally inconsistent, prioritizing flexibility and performance in the transaction confirmation and block creation processes.
The utilization of UDP/Datagram helps reduce latency and reinforce the asynchronous processing capability; however, it may imply a trade-off with data consistency. This may reflect a careful balance between consistency and performance considerations within the architecture of the StreamBase network.
